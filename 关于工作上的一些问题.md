# 1.体系文件

## 1.1 PPAP资料(Production Part Approval Process)

PPAP3需提交的18项文件：

1. **设计记录**：包含客户提供的产品图纸、3D 模型、BOM 表等核心文件，需清晰标注尺寸公差、材料标准、性能指标，若有版本迭代需标注对应版本信息。
2. **工程变更文件**：若产品设计或生产工艺有变更，需提交经客户批准的工程变更通知单（ECN），以及变更后的验证报告，证明变更后产品仍符合要求。
3. **顾客工程批准**：针对特殊设计、材料选型等需客户提前确认的内容，提交客户签署的批准文件；若客户无特殊要求，此文件可省略。
4. **设计失效模式及后果分析（DFMEA）**：记录产品设计阶段的风险识别结果，比如结构强度不足、尺寸干涉等潜在问题，同时明确对应的预防和控制措施。
5. **过程流程图**：详细绘制产品从原材料投入到成品产出的完整生产流程，标注各工序的顺序、使用设备及关键控制点，让客户清晰了解生产逻辑。
6. **过程失效模式及后果分析（PFMEA）**：聚焦生产过程中的风险，如焊接虚焊、注塑缺料等工序问题，分析失效原因和影响，并制定优化工艺参数等管控方案。
7. **控制计划**：基于 DFMEA 和 PFMEA 的结果制定，明确各工序的控制方法，例如关键尺寸的测量频率、过程参数的监控标准、不合格品处理流程等，是量产质量管控的核心依据。
8. **测量系统分析（MSA）报告**：验证卡尺、光谱仪等检测设备的可靠性，以及检测人员操作的重复性、再现性，确保测量数据的准确性，常见如量具 GR&R 分析报告。
9. **全尺寸测量结果**：从试生产批次中抽取代表性样品（通常 30 件以上），对图纸标注的所有尺寸进行 100% 检测并记录数据，证明产品尺寸符合设计规范。
10. **材料 / 性能测试报告**：材料方面需提供原材料材质单、环保认证（如 RoHS）等；性能方面需按客户标准完成耐温、耐压、疲劳等测试，附测试数据和设备校准证书。
11. **初始过程研究**：即 SPC 报告，通过分析试生产数据计算 Ppk、Cpk 等指数（通常要求≥1.33），证明生产过程稳定，能够持续产出合格产品。
12. **外观批准报告（AAR）**：针对有颜色、纹理、表面光泽等外观要求的产品，提交附带标准色卡、缺陷判定标准的报告，以及经客户确认的外观样品相关文件。
13. **生产件样品**：提交试生产阶段产出的样品，数量按客户要求（通常 3 - 5 件），样品需与量产条件一致，且标注批次、生产时间等信息。
14. **标准样品**：经客户认可的合格样品，用于后续量产过程中的外观、尺寸比对，需标注批准日期和保存要求，作为后续生产的参考基准。
15. **检查辅具**：若生产中使用专用检具、治具等检测工具，需提交辅具的设计图纸、校准报告，证明辅具精度符合检测要求，且在有效校准周期内。
16. **零件提交保证书（PSW）**：核心汇总文件，包含供应商及产品基础信息、提交等级、关键测试结果摘要，需供应商授权人签署，声明提交资料真实有效，量产条件与试生产一致。
17. **合格实验室文件**：若测试由第三方或客户指定实验室完成，需提供该实验室的 ISO 17025 认证等资质文件，证明测试结果具备权威性。
18. **客户特殊要求（CSR）的符合性证据**：针对客户提出的包装标准、标识规范、特殊认证等要求，提交对应的验证文件，如包装测试报告、行业专项认证证书等。

# 2.材料

## 2.1 低碳钢与不锈钢

低碳钢和不锈钢的主要区别体现在**化学成分**、**耐腐蚀性**、**力学性能**和**应用领域**等方面

**低碳钢**的含碳量**0.1%~0.25%**，主要成分为铁和碳，**不含鉻、镍**等合金元素
**不锈钢**含碳量**通常更低**（部分类型可能超过0.25%），**必须含有至少10.5%的铬**，**并可能添加镍、钼等**合金元素以增强性能

### 2.1.1 耐腐蚀性

不锈钢通过**铬元素**形成氧化铬保护层，耐酸碱和氧化环境腐蚀性能显著优于低碳钢
低碳钢暴露在潮湿环境中易生锈，需通过镀锌或涂漆进行防护

3.右旋螺纹和左旋螺纹

- 右旋：顺时针旋转时拧紧（默认类型，不标注）；
- 左旋：逆时针旋转时拧紧（标注 “LH”，如 M12×1.5 LH）

### 4.2.2 关键尺寸参数

1.大径 D/d

2.小径 D1/d1

3.中径 D2/d2

4.螺距 P

```
中径=大径-0.6495×螺距
小径=大径-1.0825×螺距
```



### 4.2.2 公差配合

1.公差带

```
定义：螺纹中径（或大径、小径）的允许偏差范围（由 “基本偏差” 和 “公差等级” 组成）。
基本偏差：决定公差带的位置（外螺纹用小写字母 e、f、g、h 表示，h 为上偏差 0；内螺纹用大写字母 G、H 表示，H 为下偏差 0）。
公差等级：决定公差带的宽度（数字越小精度越高，常用 3-9 级，如 6H、6g）
```

2.配合等级

```
6H/6g：中等配合（默认），互换性好，适用于一般机械连接；
6H/5g6g：高精度配合，用于精密设备、仪器；
7H/8g：松配合，用于高温、振动或装配难度大的场景。
```
